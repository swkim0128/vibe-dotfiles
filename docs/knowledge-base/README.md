# Knowledge Base (kb/)

작업 중 학습한 지식을 누적·재사용하기 위한 디렉토리.

## 목적
- 새로운 작업 시작 전(GROUND): 관련 `kb_*.md` 사전 조회로 과거 패턴 재사용
- 작업 완료 후(VERIFY/ADAPT): 새로 알게 된 사용법·에러·환경 설정 누적 기록

## 양식 (4섹션 엄격 유지)
```
## 현상
## 원인
## 해결 코드
## 향후 주의사항
```

## 파일 명명
- 주제별 prefix: `kb_<도메인>.md` (예: `kb_iconv_cache.md`, `kb_notion.md`, `kb_mac_system.md`)
- 신규 주제는 새 파일, 기존 주제는 추가 누적

상세 운영 룰은 `~/.claude/CLAUDE-user.md` 의 "지식 자동화 루프 (Knowledge Base SOP)" 섹션 참조.
